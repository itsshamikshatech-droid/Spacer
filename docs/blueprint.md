# **App Name**: Spaser Hub

## Core Features:

- User Role Selection: Allow users to select their role (Space Seller, Service Provider, Space Buyer) upon registration.
- Space Listing: Enable Space Sellers to list spaces with details like location, photos, capacity, facilities, and price. Integrate Google Maps API for location input and display.
- Service Provider Registration: Allow Service Providers to register, choose a category (Wi-Fi, catering, cleaning, AV Tech, etc.), set charges, availability, and area of operation.
- Space Search and Filtering: Enable Space Buyers to search and filter spaces based on location, purpose, date, required services, price, and ratings.
- AI Recommendation Engine: Use AI to suggest the best-matched spaces and services based on user history, budget, location, and reviews.
- Booking and Payment: Enable Space Buyers to book spaces, add optional services, and make payments through Razorpay API. Display transparent pricing.
- Conversational Interface: Implement a conversational interface between buyers and sellers with the capacity to determine the appropriate use of a maps display and links to available payment portals using a tool.

## Style Guidelines:

- Primary color: Dark blue (#1A56DB) to convey trust and professionalism.
- Background color: Light gray (#F9F9F9) to provide a clean and modern look.
- Accent color: Teal (#6EE0FF) for highlights and call-to-action buttons, inspired by a modern and vibrant feel.
- Body and headline font: 'PT Sans', a humanist sans-serif, is recommended for its modern look combined with warmth and readability. Use for both headlines and body text.
- Use clean and modern icons to represent different space types, services, and amenities.
- Employ a clean and intuitive layout with clear filtering options and prominent call-to-action buttons.
- Incorporate subtle animations to enhance user experience, such as smooth transitions and loading indicators.
